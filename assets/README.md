## Icon Requirements

For the final application icon:
- **Size**: 1024x1024 pixels minimum
- **Format**: PNG with transparency
- **Style**: Modern, clean design representing speech/microphone
- **Colors**: Compatible with macOS menu bar (works in both light/dark modes)
